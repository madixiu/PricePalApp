# PricePal
---
**PricePal** is a versatile mobile application designed to provide real-time price information for groceries, cryptocurrencies, and various currencies from around the world. 
With PricePal, users can easily track and compare prices for everyday items, monitor cryptocurrency values, and stay updated on exchange rates for different currencies.
_Key Features:_
* Grocery Prices: Access up-to-date information on the prices of groceries to make informed shopping decisions.
* Cryptocurrency Prices: Stay informed about the latest prices and trends in the cryptocurrency market.Currency Converter: Convert between different currencies with accurate exchange rate information.
* Multi-Currency Support: View prices in various currencies to facilitate international comparisons.
